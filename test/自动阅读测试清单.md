# 阶段三 自动阅读 — 测试验收清单

## 一、UI 展示

- [ ] **1.1** 点击悬浮按钮 → 阅读设置，弹窗中出现「📜 自动阅读」区域
- [ ] **1.2** 初始状态：「▶ 开始自动阅读」按钮 + 「↓ 向下」高亮 + 速度滑块默认值 50
- [ ] **1.3** 弹窗在暗黑/护眼主题下，自动阅读控件颜色跟随主题变量

## 二、开始 / 暂停

- [ ] **2.1** 点击「▶ 开始自动阅读」，按钮立即变为「⏸ 暂停自动阅读」并高亮
- [ ] **2.2** 点击后内容开始向下滚动
- [ ] **2.3** 滚动中再次点击按钮，滚动停止，按钮恢复「▶ 开始自动阅读」

## 三、速度调节

- [ ] **3.1** 拖动速度滑块，数值实时变化（10–100）
- [ ] **3.2** 松手后滚动速度改变（数值越大越快）
- [ ] **3.3** 滚动中调节速度，滚动不中断，速度即时生效
- [ ] **3.4** 点击快捷速度按钮（20/40/60/80），滑块和数值同步更新，速度立即生效

## 四、方向切换

- [ ] **4.1** 点击「↑ 向上」按钮，按钮高亮切换，「↓ 向下」取消高亮
- [ ] **4.2** 停止状态下切换方向，点击开始后按新方向滚动
- [ ] **4.3** 滚动中切换方向，立即改为新方向继续滚动

## 五、键盘快捷键

- [ ] **5.1** 按 `空格` 键 → 开始自动阅读
- [ ] **5.2** 滚动中按 `空格` → 暂停
- [ ] **5.3** 在输入框（搜索框、笔记等）内按空格，**不触发**自动阅读

## 六、数据持久化

- [ ] **6.1** 设置速度 80、方向「↑」，关闭弹窗 → 刷新页面 → 重新打开阅读设置，速度和方向保持
- [ ] **6.2** 滚动中刷新页面 → 自动停止，状态恢复了但不自动开始

## 七、恢复默认

- [ ] **7.1** 自动阅读运行中 → 菜单点击「🔄 恢复默认设置」→ 滚动停止，速度和方向重置为 50 / 向下
- [ ] **7.2** 恢复默认后屏占比、主题也一并重置，互相不干扰
