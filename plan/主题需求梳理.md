# 主题颜色需求梳理

## 一、用户期望

插件提供 **3 种主题**：

| 主题 | 背景色 | 文字色 | 风格 |
|-----|--------|--------|------|
| ☀️ 明亮 | 纯白 `#ffffff` | 纯黑 `#000000` | 类似纸张阅读 |
| 🌙 暗黑 | 深灰 `#1e1e1e` | 浅灰 `#e8e8e8` | 夜间护眼 |
| 👁️ 护眼 | 米黄 `#f5e6c8` | 深棕 `#1a0a00` | 长时间阅读舒适 |

---

## 二、微信读书官方主题机制

微信读书通过 **`body.wr_whiteTheme`** 这个 CSS 类控制全局主题：

- **有 `wr_whiteTheme`** → 白天模式（浅色背景 + 深色文字）
- **无 `wr_whiteTheme`** → 夜间模式（深色背景 + 浅色文字）

`wr_whiteTheme` 由 React 主动管理，通过 `classList` 增删触发全局样式变更。

---

## 三、当前约束

### 已确认事实

- 微信读书官方主题仍由 React 内部状态管理，单纯改 `body.wr_whiteTheme` 不足以稳定切换官方主题。
- 在当前页面环境下，程序化点击官方主题按钮也不能稳定触发官方夜间/白天切换。
- 因此，**插件主题与官方主题不再做强制联动**，当前目标只保留一件事：插件自己的阅读配色必须稳定、清晰、可持久化。

### 当前策略

- 插件自己控制背景色
- 插件自己控制正文文字色
- 插件不再承诺“自动切官方夜间/白天”

---

## 四、当前最终方案：插件独立控制背景与文字

### 核心思路

**插件直接控制阅读页的背景和正文文字，不再依赖官方主题状态。**  
通过 `<style>` 注入控制背景，再通过 `element.style.setProperty(..., 'important')` 直接控制正文文字颜色，确保三种主题下都能清晰可读。

### 映射关系

| 插件主题 | 插件设背景 | 插件设文字色 | 是否强制同步官方主题 |
|---------|---------|------------|:---:|
| ☀️ 明亮 | `#ffffff` | `#000000` | 否 |
| 🌙 暗黑 | `#121212` | `#ffffff` | 否 |
| 👁️ 护眼 | `#f5e6c8` | `#1a0a00` | 否 |

### 四层机制

#### 第 1 层：`wreThemeCSS` 负责背景
- 通过 `<style id="we-read-enhancer-style">` 注入到 `<head>`
- 为阅读页主要容器统一设置背景色
- 主题切换时重建 `<style>`，保证背景立即切换

#### 第 2 层：`applyThemeColors()` 负责正文文字
- 遍历所有 `.readerChapterContent`
- 对正文容器及其子元素直接写入 `color` 和 `-webkit-text-fill-color`
- 避免双页/预渲染模式下只有部分正文被正确上色

#### 第 3 层：`MutationObserver` 负责新节点补色
- 监听 `.app_content_in_reader`
- 阅读内容刷新、分页或重渲染后，继续补刷当前主题文字颜色

#### 第 4 层：初始化恢复已保存主题
- `init()` 中读取已保存主题
- 刷新页面后自动恢复插件主题
- 不再尝试同步官方白天/夜间状态

### 适用场景总览

| 操作 | 结果 |
|------|------|
| 点插件"暗黑" | 背景深灰，正文浅色清晰 |
| 点插件"明亮" | 背景全白，正文深色清晰 |
| 点插件"护眼" | 背景米黄，正文深色清晰 |
| Ctrl+F5 刷新 | `init()` 自动恢复上次保存的插件主题 |

### 用户提示

主题设置面板底部显示灰字提示：
> 📌 插件主题只控制本插件的阅读配色，不再强制同步微信读书官方白天/夜间模式

## 五、代码实现要点

| 文件 | 位置 | 内容 |
|------|------|------|
| `content.js` | `wreThemeCSS` | 三个主题的背景 CSS |
| `content.js` | `applyTheme()` | 切换插件主题类并重建 `<style>` |
| `content.js` | `applyThemeColors()` | 直接给正文和子元素写入文字颜色 |
| `content.js` | `init()` | 恢复已保存主题 |
| `content.js` | modal HTML | 主题按钮下方 `.wre-theme-hint` 提示文字 |
| `content.css` | `.wre-theme-hint` | 灰色小字样式 |
