# 阶段六 新手引导 — 测试验收清单

## 一、首次安装

- [ ] **1.1** 全新安装插件 → 加载微信读书 → 自动弹出「欢迎使用微信读书增强插件」面板
- [ ] **1.2** 面板标题为「欢迎使用微信读书增强插件」
- [ ] **1.3** 面板内容包含功能介绍文字 + 快捷键速查表（Space / D / F / ?）
- [ ] **1.4** 面板底部显示「更多功能请点击左下角悬浮图标 → 查看菜单」

## 二、弹窗关闭

- [ ] **2.1** 点击右上角 `×` 按钮 → 关闭欢迎面板
- [ ] **2.2** 点击面板外部遮罩区域 → 关闭欢迎面板

## 三、刷新后不再弹出（同版本）

- [ ] **3.1** 首次弹出后关闭面板 → 刷新页面（F5 / Ctrl+F5） → 不再弹出欢迎面板
- [ ] **3.2** 再次打开微信读书新标签页 → 不再弹出欢迎面板

## 四、版本升级后重新弹出

- [ ] **4.1** `manifest.json` 版本号变更（如 0.7.0 → 0.8.0） → 重新加载扩展 → 再次弹出欢迎面板
- [ ] **4.2** 升级弹出后关闭面板 → 刷新页面 → 不再重复弹出（同版本逻辑）

## 五、快捷键帮助面板联动

- [ ] **5.1** 欢迎面板关闭后 → 按 `?` → 弹出快捷键帮助面板 → 内容与欢迎面板快捷键一致
- [ ] **5.2** 欢迎面板中点击 `×` 关闭 → 悬浮菜单中点击「⌨️ 快捷键说明」→ 正常打开快捷键帮助面板

## 六、主题兼容

- [ ] **6.1** 暗黑主题下弹出欢迎面板 → 面板文字/背景随主题正常显示
- [ ] **6.2** 护眼主题下弹出欢迎面板 → 面板文字/背景随主题正常显示
- [ ] **6.3** 欢迎面板打开期间切换主题 → 面板即时跟随主题变化

## 七、异常场景

- [ ] **7.1** chrome.storage.local 不可用时 → 插件正常加载，不报错（每次仍弹出欢迎面板属可接受降级）
- [ ] **7.2** 欢迎面板不阻塞其他快捷键（D / F / Space 在面板打开时仍正常生效）
