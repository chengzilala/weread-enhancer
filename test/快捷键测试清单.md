# 阶段四 快捷键 — 测试验收清单

## 一、← / → 翻页

- [ ] **1.1** 阅读页面按 `←`，翻到上一页（微信读书原生支持，插件不拦截）
- [ ] **1.2** 阅读页面按 `→`，翻到下一页（微信读书原生支持，插件不拦截）
- [ ] **1.3** 滚动模式 + Canvas 模式下 `←`/`→` 均正常翻页
- [ ] **1.4** 在输入框（搜索框、笔记等）内按 `←`/`→`，仅移动光标，不翻页（原生行为）

## 二、Space 自动阅读

- [ ] **2.1** 按 `空格` → 开始自动阅读，UI 按钮同步变为「⏸ 暂停自动阅读」
- [ ] **2.2** 自动阅读运行中按 `空格` → 暂停，UI 按钮恢复「▶ 开始自动阅读」
- [ ] **2.3** 在输入框内按 `空格`，**不触发**自动阅读（仅输入空格字符）

## 三、D 勿扰模式

- [ ] **3.1** 默认状态下按 `D`，悬浮按钮变为透明隐藏（鼠标悬停才显示）
- [ ] **3.2** 勿扰模式下再次按 `D`，悬浮按钮恢复正常显示
- [ ] **3.3** 按 `D` 切换后刷新页面，勿扰状态保持
- [ ] **3.4** 小写 `d` 与大写 `D` 均能触发
- [ ] **3.5** 在输入框内按 `D`，**不触发**勿扰切换（仅输入字母）
- [ ] **3.6** 勿扰模式下鼠标悬停图标区域 → 图标淡入显示 → 移开鼠标 → 图标淡出隐藏
- [ ] **3.7** 勿扰模式下点击图标 → 退出勿扰模式，悬浮按钮恢复正常显示，同时打开菜单

## 四、? 快捷键帮助面板

- [ ] **4.1** 按 `?` → 弹出「⌨️ 快捷键说明」弹窗
- [ ] **4.2** 弹窗内展示完整快捷键表格：←/→、Space、D、?
- [ ] **4.3** 帮助面板可见时再次按 `?` → 关闭弹窗
- [ ] **4.4** 点击弹窗右上角 `×` 按钮 → 关闭弹窗
- [ ] **4.5** 点击弹窗外部遮罩区域 → 关闭弹窗
- [ ] **4.6** 暗黑/护眼主题下帮助面板颜色跟随主题
- [ ] **4.7** 从悬浮菜单「⌨️ 快捷键说明」点击也能打开此面板
- [ ] **4.8** 在输入框内按 `?`（需 Shift），**不触发**帮助面板

## 五、键盘事件安全性

- [ ] **5.1** 程序派发的键盘事件（isTrusted=false）**不触发**任何快捷键
- [ ] **5.2** 自动阅读 Canvas 模式派发的 ArrowDown/ArrowUp 不误触发快捷键
- [ ] **5.3** 快捷键与其他浏览器扩展无冲突

## 六、多快捷键协同

- [ ] **6.1** 自动阅读运行中按 `D` 切换勿扰，自动阅读不中断
- [ ] **6.2** 帮助面板打开时其他快捷键仍正常生效

## 七、数据持久化

- [ ] **7.1** 开启勿扰模式 → 刷新页面 → 勿扰状态保持
- [ ] **7.2** 切换主题 → 刷新页面 → 主题保持
- [ ] **7.3** 自动阅读状态 → 刷新页面 → 停止且不自动恢复（预期行为）

## 八、F 全屏模式

- [ ] **8.1** 阅读页面按 `F` → 浏览器进入全屏，屏占比自动设为 100%
- [ ] **8.2** 全屏状态下按 `F` → 退出全屏，屏占比恢复到进入前的值
- [ ] **8.3** 全屏状态下按 `Esc` → 退出全屏，屏占比自动恢复
- [ ] **8.4** 屏占比 70% 时按 `F` 进入全屏 → 显示 100% → 按 `F` 退出 → 恢复 70%
- [ ] **8.5** 小写 `f` 与大写 `F` 均能触发
- [ ] **8.6** 在输入框内按 `F`，**不触发**全屏切换
- [ ] **8.7** 全屏状态下，悬浮按钮自动隐藏（勿扰模式开启）
- [ ] **8.8** 退出全屏后，勿扰状态恢复到进入前（进入前关闭 → 退出后关闭；进入前已开启 → 退出后仍开启）
